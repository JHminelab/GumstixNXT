NXT Sense is a NXT sensor driver framework, which purpose is to ease the development of drivers. It provides a way to handle which NXT port a sensor is connected to and allows the user to dynamically setup and (re)configure which NXT sensors are connected to each port.
For more specific use and information read the provided report (and make a better user guide): NXT_Sense.pdf

Note:
Depending on your setup the makefiles can require modifications. But they should work with the setup made from following the guide for setting up openembedded (as of current writing).
